# [go-init]
项目介绍：

[go-init]是一个基于Go的Web框架，提供了一系列实用的Web开发功能，如路由、模板引擎、静态文件服务、数据库支持等。

功能
路由：支持多种路由模式，如基于正则表达式、基于静态文件目录等。
模板引擎：支持多种模板引擎，如Go模板、Mustache等。
静态文件服务：支持静态文件目录配置，提供HTTP服务。
数据库支持：支持多种数据库类型，如MySQL、PostgreSQL等。
